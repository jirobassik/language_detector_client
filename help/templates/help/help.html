Инструкция по использованию продукта

Функция 1: Загрузка файла

Следуйте этим инструкциям для Загрузки файла:

  1. Перейдите на страницу для работы с файлами, для этого нажмите кнопку архива.
  2. Нажмите кнопку выбрать файл
  3. Выберите файл в файловом менеджере формата .pdf.
  4. Нажмите кнопку загрузить файл.

Функция 2: Определение языка файла

Произведите следующие действия для определения языка файла:

  • На главной странице определитесь с файлом для определения языка.
  • В файловой карточке на выбор есть три метода определения языка: алфавитный, коротких слов, нейросетевой.
  • Выберите один из вариантов и будет выдан результат.

Функция 3: Дополнительные функции

  1. Для удаления файла, нажмите кнопку с корзиной.
  2. Для сохранения файла на ваше устройство, нажмите кнопку с загрузкой.
